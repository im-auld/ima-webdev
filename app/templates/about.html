About Me

My name is Ian Michael Auld (get it IMA Webdev), I'm 28 years old and have been using Python for about ~1-1.5 years. I am a Marine veteran that completed one tour in support of Operation: Iraqi Freedom. I was working primarily as the shipping manager for a small to mid size retail website for the past 4 years. I fulfilled various other roles while working there which lead me to begin learning programming to help solve a problem I encountered.

I started learning C# initially which can be quite cumbersome at times but one day while shopping in Barnes and Noble with my girlfriend I picked up a Python book on a whim, remembering this XKCD comic. Since then I have made a few apps with Python mostly using Flask (including this website).

I have a solid foundation in Python as well as strong understanding of HTML and CSS as well as a working knowledge of Flask, SQL, Javascript and jQuery. I am currently attending Code Fellows in Seattle, WA which I will be completing at the end of July. You can see my learning journal, an ongoing project I'll be working on here.
