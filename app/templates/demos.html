Datamapper

Datamapper is a Flask application that uses economic data collected by the federal government to generate heat maps. You can then view the changes in that data by adjusting the slider. Not all data sets are complete and there isn't always data for every state. The front uses jQuery/JS, a library called heatmap.js for the heat maps as well as the Bootstrap CSS framework.

Flask Inventory

Flask Inventory is a lightweight inventory management system suited best for small inventories of non-serialized items of a trivial cost. It is simple to use and has very low overhead. This project was a proof of concept for me, that Flask could be used to create more extensive projects than simple one page apps. This is (and probably always will be ) a work in progress.

tweetTrack

tweetTrack is Flask application that uses a users tweet history to try and predict their location. This application was inspired by a research paper written by IBM. By analyzing the twitter histories of users from the 100 most populace cities in the US a statistical model was built. This model helped to predict where a user was (or at least where they tweet about).
